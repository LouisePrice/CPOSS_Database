# Other notes

There was one search done with CrystalPredictor (Study_ID=20).  The conformation was created with no induction.  The structures were then reminimized:

- “withCT” which had the same conformation but different packing and energy (Study_ID=21)

- with induction, which had the same conformation and packing but different energy (Study_ID=50)

- with CrystalOptimizer, which allowed the conformation and packing to change (Study_ID=10)
